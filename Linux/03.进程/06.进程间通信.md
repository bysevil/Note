本质：让多个进程访问同一片空间
## 管道
管道由操作系统提供，本质是一个文件。这个文件保存在内存上，而不是磁盘。

管道是单向通信的，一个出口一个入口。本质上，是通过两个进程分别拿到管道文件的读权限，一个拿到写权限。

创建管道
```C++
int pipefd[2] = {0,0};
int pipe(int pipefd[2]);

pid_t pid = fork();

if(pid == 0){
	close(pipefd[1]);
}else{
	close(pipefd[0]);
}
```
pipefd是输出型参数，pipefd\[0\] 是管道 pipefd\[1\]是管道入口。
