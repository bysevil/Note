冒泡排序是交换排序的一种

时间复杂度：$O(n^2)$
空间复杂度：$1$
稳定性：稳定

冒泡排序每次遍历会选出一个最大的数放到数组的最后，然后在剩下的数中再选最大，直到数组有序

1. 遍历数组，但不遍历最后一个数，如果遍历到的数大于它的下个数，则交换两个数。
2. 继续遍历，但不遍历最后两个数。
3. 继续遍历，直到数组有序

```c
void BubbleSort(int* nums, int numsSize){
    for(int i = numsSize; i > 0;i--){
        for(int j = 1; j < i; j++){
            if(nums[j-1] > nums[j])
                swap(nums+j-1,nums+j);
        }
    }
}
```

时间复杂度： $O(n^2)$

空间复杂度：$O(1)$

稳定性：稳定
