## 忽略文件

你可以忽略指定的文件，使他不被git追踪

这个忽略通过 `.gitignore`文件实现，在 `.gitignore`中写明的文件将会被git忽略

`.gitignore`文件必须存放在仓库根目录。

`.gitignore`文件本身可以被推送。

## 忽略文件

你可以指定忽略某个文件

```shell
文件名.文件后缀
```

你可以指定忽略某个后缀的文件

```shell
*.文件后缀
```

> 每行书写一个文件，或者一类文件，无需分号等其他内容。

## 强制添加文件

被忽略的文件也可以被强制添加

```shell
git add -f 文件名
```

> 一般不使用，不要破坏文件的规则

## 指定不忽略文件

我们可以指定不忽略某些文件。

```shell
!文件名.文件后缀
```

> 比如说你指定了忽略所有的 `.c`文件，但你不想忽略 `test.c`
>
> ```shell
> *.c
> !test.c
> ```

我们可以查看自己是否忽略或者不忽略了哪些文件

```shell
git check-ignore -v 文件名.文件后缀
```
